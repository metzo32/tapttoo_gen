# Tapttoo
## 커스텀 타투 도안 제작 서비스



## 프로젝트 개요

사용자가 원하는 키워드를 입력하면 추상화하여 세상에 하나 뿐인 타투 도안으로 만들어주는 서비스입니다.

### 주요 기능

+ 반응형 디자인
+ DALL-E API를 활용하여 사용자의 프롬프트를 입력받고, 기존에 설계해둔 프롬프트와 연결하여 이미지 생성
+ 흑백모드 및 컬러모드 도안 생성 기능
+ 저장 및 리셋 기능
+ 불가피하게 발생하는 10초 딜레이 동안 프로그레스 애니메이션 재생
+ 사이드바 및 Darkmode/Lightmode 토글
